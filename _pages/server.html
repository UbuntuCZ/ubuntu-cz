Ubuntu Server na architektuře Power

Kromě platforem x86 a ARM podporuje Ubuntu také architekturu Power. Pro podniková datacentra to znamená, že nyní můžete postavit svou infrastrukturu na libovolném preferovaném hardwaru.

  • Výkon a mnohoúčelnost

    Není důležité, jestli chcete rozvinout NoSQL databázi, webovou farmu nebo cloudové zázemí, Ubuntu má všechny potřebné vlastnosti které potřebujete. Ubuntu je certifikováno vedoucími OEM výrobci hardwaru, což vám s komplexními implementačními nástroji umožní získat z vaší infrastruktury co nejvíce.

  • Výkonná, bezpečná a všestranná technologie pro rychle rostoucí podniky.

    Pravidelný vývojový cyklus Ubuntu zajišťuje podporu nejnovějších aplikací. Tenká instalace a integrovaná implementace systému spolu se snadnými postupy dělají Ubuntu Server skvělé řešení pro snadné zavádění a následnou správu i v rozsáhlém měřítku.
